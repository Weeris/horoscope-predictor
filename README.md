# Multi-System Horoscope Predictor

A comprehensive horoscope application that combines multiple divination systems to provide daily, weekly, and monthly predictions based on your birth date.

## Features

- **Multiple Divination Systems**:
  - Chinese Zodiac (Year-based)
  - Western Astrology (Sun Sign)
  - Numerology (Life Path Number)
  - Moon Sign (Calculated from birth date)
  
- **Prediction Types**:
  - Daily Horoscope
  - Weekly Forecast
  - Monthly Outlook
  
- **User-Friendly Interface**:
  - Simple date input
  - Color-coded predictions
  - Detailed explanations

## Technologies Used

- Streamlit (for web interface)
- Python date/time calculations
- Custom prediction algorithms
- Plotly for visualization

## Installation

```bash
pip install streamlit pandas numpy
```

## Usage

Run the application:
```bash
streamlit run app.py
```
